图片验证码原理： 服务器生成包含随机字符的字符串图片发给客户端，客户端提交数据需要填写字符串作为验证。而字符串保存在图片中，机器难以识别，达到防止有人利用程序恶意发送信息的目的。

JAVA随机生成图片

JAVA随机生成图片步骤如下：

  1. 创建图片缓存区 传参为宽高和图片类型

  2. 获取画笔并绘画

  3. 输出图片
